# torytis 버전 업데이트

`@wisdomstar94/torytis` 패키지 종속성 버전을 업데이트 하시면 그에 따라 프로젝트 내에도 어떤 파일을 추가하거나, 삭제하거나, 수정해야 할 수도 있습니다.
최신버전에서 요구하는 프로젝트 폴더 내의 파일 구조가 어떻게 되는지는 [torytis-start-template](https://github.com/wisdomstar94/torytis/tree/main/src/packages/torytis-start-template) 에서 확인하실 수 있으시며,
필요한 경우 `git clone https://github.com/wisdomstar94/torytis.git` 으로 직접 torytis 를 내려받아 `src/packages/torytis-start-template` 경로를 확인해보실 수도 있겠습니다.
