# torytis 개발 결과물 미리보기

torytis 에서는 스킨 개발 결과를 티스토리에 직접 업로드 하지 않아도 로컬 환경에서 확인해 볼 수 있는 `dev server` 기능을 제공합니다. 아래 명령어를 입력하시면 됩니다.

```bash
npm run dev
```

이후 `http://localhost:3000` 에 접근하면 확인 가능합니다.
소스 코드의 내용이 변경될 경우 브라우저 화면에 자동으로 반영되는 기능도 적용되어 있습니다.

## dev server 에서 제공하는 미리보기 페이지

dev server 에서 제공하는 페이지들의 url 은 실제 티스토리 url 과 패턴이 동일하며 아래와 같은 페이지들을 제공하고 있습니다.

| url                                           | 설명                    |
| --------------------------------------------- | ----------------------- |
| `/`                                           | 홈 post index           |
| `/category`                                   | 전체분류보기 post index |
| `/category/[부모카테고리명]`                  | 부모카테고리 post index |
| `/category/[부모카테고리명]/[자식카테고리명]` | 자식카테고리 post index |
| `/notice`                                     | 공지사항 post index     |
| `/notice/[id]`                                | 공지사항 post permalink |
| `/guestbook`                                  | 방명록                  |
| `/[id]`                                       | 글 post permalink       |
| `/tag`                                        | 태그 목록               |
| `/search/[string]`                            | 검색 post index         |

치환자도 실제 티스토리에 스킨을 적용했을 때와 동일하게 적용됩니다.

## 개발용 데이터

`npm run dev` 로 로컬에서 스킨 결과물을 볼 때 표시되는 데이터 들은 `config/torytis-dev.config.json` 파일에서 확인 및 수정하실 수 있습니다.

## 실제 티스토리 결과와의 차이

torytis 에서 제공하는 `dev server` 는 티스토리에 스킨을 적용했을 때 치환되는 부분을 최대한 동일하게 구현하고자 하였으나 일부분에 있어서는 아직 지원되지 않는 부분이 있으며, 일부는 실제 티스토리와는 다르게 치환되는 부분도 존재할 수 있습니다.

해당 부분은 발견하는 즉시 수정하고자 합니다만, 미처 제가 발견하지 못한 부분이 있을 수 있습니다. 만약 여러분들이 torytis 의 `dev server` 를 사용하시다가 실제 티스토리에서의 결과와 `dev server` 에서의 결과가 다른 부분이 발견되시면 torytis 의 [issues](https://github.com/wisdomstar94/torytis/issues) 탭에 등록해 주시면 확인해보도록 하겠습니다.
