# torytis 개발 결과물 미리보기

torytis 에서는 스킨 개발 결과를 티스토리에 직접 업로드 하지 않아도 로컬 환경에서 확인해 볼 수 있는 기능을 제공합니다. 아래 명령어를 입력하시면 됩니다.

```bash
npm run dev
```

이후 `http://localhost:3000` 에 접근하면 확인 가능합니다.

현재 해당 기능은 티스토리에서 치환되는 부분을 100% 모두 완벽하게 구현하지는 못했기 때문에 npm run dev 의 결과와 실제 티스토리에 올렸을 때의 결과에 차이가 발생할 수 있습니다. 그러므로 npm run dev 는 반드시 참고 용도로만 사용하시고 실제 결과는 반드시 티스토리에 올린 후 티스토리에서 확인해주셔야 합니다.
본 기능을 좀 더 고도화 하면서 로컬 환경의 결과와 실제 티스토리에서의 결과가 차이가 많이 없게 하기 위해 노력할 예정입니다.

또한 현재는 코드가 변경되었을 경우 곧바로 브라우저에 변경 사항이 반영되는 핫리로드 기능이 적용되어 있지 않은 상태이므로 코드에 변경 사항이 생겼을 경우 npm run dev 을 입력했던 터미널에서 현재 세션을 종료한 후 다시 npm run dev 를 입력해주셔야 합니다.

> 현재 npm run dev 에 대한 기능 고도화를 염두에 두고 있습니다. (로컬환경과 티스토리에서의 스킨 결과물에 대한 차이 줄이기, 핫리로드 적용 등)
