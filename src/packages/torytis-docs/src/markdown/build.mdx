# torytis 빌드

스킨 개발이 모두 완료되었거나 중간에 실제로 티스토리에 반영해서 결과를 확인해보고 싶을 경우 빌드를 할 수 있습니다. 아래 명령어를 입력하면 빌드가 진행됩니다.

```bash
npm run build
```

## 빌드 결과물

빌드 후 결과물은 모두 아래 예시 사진 처럼 `.torytis/` 폴더 밑으로 생성됩니다.

티스토리에 스킨을 업로드 할 때 `.torytis/` 폴더 밑에 존재하는 모든 파일들을 모두 올리면 됩니다.

## 빌드시 .torytis 에 생성되는 것들

- `src/index.component.tsx` 파일이 번들되어 생성된 `skin.html`, `style.css` 파일
- `src/**/*.script.tsx` 파일들이 번들되어 생성된 `script.js` 파일
- `src/public/` 폴더 밑에 존재하는 모든 파일
