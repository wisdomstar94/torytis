# torytis 란?

티스토리 스킨 개발을 좀 더 편하게 할 수 있도록 도와주는 rust 로 만들어진 프레임워크 입니다.

# torytis 를 개발하게 된 이유

티스토리 스킨 개발 시 다음과 같은 아쉬운 점이 있었습니다.

1. `skin.html` 파일에 모든 페이지에 필요한 모든 요소를 전부 작성하는 구조이므로 한 파일에 내용이 길어지고 그 만큼 개발 및 유지보수에 아쉬움이 있었습니다.
2. 개발 결과를 확인해야 할 때마다 티스토리 블로그 관리 설정 페이지에 들어가 스킨을 업로드 하고 적용 해야 하는 과정이 필요했습니다.
3. 티스토리에서 요구하는 일부 치환자 같은 경우는 html 에 그대로 작성하게 되면 vscode 등에서 문법 에러로 인지하고 문제라고 표시합니다.
4. 변수 치환자나 변수 태그 등을 작성할 때 오타 여부를 알기가 어려웠습니다.
5. 별도 번들러 없이 `*.html`, `*.css`, `*.js` 파일로만 개발하다보니 npm 패키지를 이용하는데 불편함이 있었고 `typescript` 를 사용할 때의 타입 추론 및 타입 자동 완성 목록 제공도 받지 못해 아쉬웠습니다.

그래서 이와 같은 아쉬웠던 점들을 해소하고 싶어서 티스토리 개발에 최적화된 프레임워크를 직접 개발하였습니다.
